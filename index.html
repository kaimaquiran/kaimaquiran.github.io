Ad Astra: the Benildean Yearbook Subscription System and BYO Admin Panel (2014)

The Ad Astra: the Benildean Yearbook Subscription System (ABYSS) is a Web Application developed for the Benildean Yearbook Office (BYO) to augment their Subscription Operations.

It's an application designed for graduating students who wish to subscribe to Ad Astra's Yearbook package and services.

The aim of this project was to simplify and automate the subscription process by eliminating the use of paper-bound application/subscription forms.
The project's scripts were written in PHP and using a MySQL database on a webhost cloud server.

Along with the Subscription System is the BYO Administrator Panel, which serves as the central hub of BYO's information system and is used exclusively by BYO staff.

The Administrator Panel currently handles two modules: Subscription (ABYSS) and Recruitment.

Kash considers this as one of his biggest projects he's been ever involved with.

The System is still in use up to this day, and is currently being improved by the incumbent Web Development team.
